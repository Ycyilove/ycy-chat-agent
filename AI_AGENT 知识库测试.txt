# AI Agent & RAG 知识库测试文档
文档分类：开发测试专用 | 创建时间：2026.04.02

## 1. 基础功能说明
1. AI对话助手支持实时聊天、角色扮演、创意文案生成
2. 系统可上传PDF/TXT/DOCX多种格式文件
3. 上传文件自动分块、向量化、存入本地FAISS向量库

## 2. RAG核心流程
文件上传 → 文本清洗 → 切片分块 → Embedding向量生成 → FAISS索引存储
用户提问 → 语义相似度检索 → 召回相关片段 → 拼接上下文 → LLM生成答案

## 3. 进阶扩展能力
- 长期会话记忆：SQLite保存session会话与聊天记录
- 工具调用：代码运行、数据解析、文件处理内置工具
- 权限隔离：不同会话知识库互相独立不混淆

## 4. 测试专用彩蛋

用来检验RAG是否能精准检索冷门专属内容！
